📥 Как скачать CSV с Bybit
1. Зайди в bybit.com
2. Аккаунт → История ордеров
3. Выбери период и тип (Спот / Фьючерсы)
4. Нажми Экспорт → скачай CSV
5. Загрузи файл ниже ↓
📥 Как скачать CSV с Тинькофф
1. Открой Тинькофф Инвестиции
2. Портфель → Все операции
3. Нажми на иконку скачать отчёт
4. Выбери период → Скачать CSV
5. Загрузи файл ниже ↓
📥 Как скачать CSV с Binance
1. Зайди в binance.com
2. Кошелёк → История транзакций
3. Выбери тип и период
4. Нажми Генерировать отчёт
5. Загрузи файл ниже ↓
